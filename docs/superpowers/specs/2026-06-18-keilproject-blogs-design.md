## 项目1：2-1 点亮一个LED

### 项目目标
点亮一个LED灯，学习GPIO端口的基本控制。

### 代码解析
```c
#include <REGX52.H>

void main() {
    P2=0x55; //01010101
    while(1)
    {
        
    }
}
```

**代码解释：**
- `#include <REGX52.H>`：包含51单片机寄存器定义头文件
- `P2=0x55`：设置P2端口为0x55（二进制01010101）
- `while(1)`：主循环，保持程序持续运行

### 关键知识点
1. GPIO端口控制
2. 十六进制与二进制转换
3. 单片机程序结构

### 遇到的问题和解决方法
**问题**：初学者可能不理解为什么使用0x55这个数值。
**解决**：0x55转换为二进制是01010101，对应P2端口的8个引脚，1表示高电平，0表示低电平。

## 项目2：2-2 LED闪烁

### 项目目标
实现LED闪烁效果，掌握延时函数的使用。

### 代码解析
```c
#include <REGX52.H>
#include <INTRINS.H>

void Delay500ms()		//@11.0592MHz
{
	unsigned char i, j, k;

	_nop_();
	i = 4;
	j = 129;
	k = 119;
	do
	{
		do
		{
			while (--k);
		} while (--j);
	} while (--i);
}


void main()
{
	while(1){
		P2=0xfe;
		Delay500ms();
		P2=0xff;
		Delay500ms();
	}
}
```

**代码解释：**
- `#include <INTRINS.H>`：包含内部函数头文件，提供`_nop_()`函数
- `Delay500ms()`：延时函数，通过循环实现约500ms延时
- `P2=0xfe`：点亮第一个LED（二进制11111110）
- `P2=0xff`：熄灭所有LED（二进制11111111）

### 关键知识点
1. 延时函数的实现原理
2. 循环嵌套控制延时时间
3. LED闪烁的逻辑控制

### 遇到的问题和解决方法
**问题**：延时时间不准确，闪烁速度与预期不符。
**解决**：通过调整循环次数来校准延时时间，使用11.0592MHz晶振时需要特定的循环参数。
