## 项目1：6-1 矩阵键盘

### 项目目标
学习矩阵键盘的扫描原理，实现按键检测和显示。

### 代码解析

**main.c（主程序）：**
```c
/**
 * 矩阵键盘密码输入实验
 *
 * CPU: 89C52
 * 晶振频率: 12MHz
 *
 * 功能描述：
 *   通过4x4矩阵键盘输入按键编号，并在LCD1602液晶屏上实时显示。
 *   第1行显示提示文字 "password:"，第2行显示最近一次按下的按键编号（1~16）。
 */

#include <reg52.h>       // 89C52 特殊功能寄存器定义
#include "LCD1602.h"     // LCD1602 驱动头文件
#include "delay.h"       // 延时函数头文件
#include "matrixkey.h"   // 矩阵键盘扫描函数头文件

unsigned char keynum;    // 存储当前检测到的按键编号（1~16，0表示无按键）

void main()
{
    LCD_Init();                     // 初始化LCD1602
    LCD_ShowString(1, 1, "password:"); // 第1行第1列显示提示文字
    while (1)                       // 主循环，持续扫描键盘
    {
        keynum = matrixkey();       // 调用矩阵键盘扫描函数
        if (keynum != 0) {          // 若有按键按下
            LCD_ShowNum(2, 1, keynum, 4); // 在LCD第2行显示按键编号
        }
    }
}
```

**代码解释：**
- `matrixkey()`：矩阵键盘扫描函数，返回按键编号（1-16）或0（无按键）
- 通过行列扫描法检测按键位置
- 使用LCD1602显示按键编号

### 关键知识点
1. **矩阵键盘原理**：通过行列交叉点检测按键，减少IO口使用
2. **行列扫描法**：逐行输出低电平，检测列线状态
3. **按键消抖**：矩阵键盘同样需要消抖处理

### 遇到的问题和解决方法
**问题**：矩阵键盘扫描不准确，有时检测不到按键。

**解决**：检查以下几点：
1. 确认行列线连接正确
2. 增加适当的延时进行消抖
3. 检查扫描函数的实现逻辑
4. 确认IO口配置正确

## 项目2：6-2 矩阵键盘密码锁

### 项目目标
使用矩阵键盘实现密码锁功能，学习实际应用开发。

### 代码解析

**main.c（主程序）：**
```c
/**
 * 矩阵键盘密码锁实验
 *
 * CPU: 89C52
 * 晶振频率: 12MHz
 *
 * 功能描述：
 *   通过4x4矩阵键盘输入4位密码，按键11确认，按键12清除。
 *   预设密码为3456，输入正确显示"OK"，错误显示"ERR"。
 */

#include <reg52.h>       // 89C52 特殊功能寄存器定义
#include "LCD1602.h"     // LCD1602 驱动头文件
#include "delay.h"       // 延时函数头文件
#include "matrixkey.h"   // 矩阵键盘扫描函数头文件

unsigned char keynum;    // 存储当前检测到的按键编号
unsigned int password;   // 存储输入的密码
unsigned int count;      // 存储已输入的密码位数

void main()
{
    LCD_Init();                     // 初始化LCD1602
    LCD_ShowString(1, 1, "Password:"); // 第1行第1列显示提示文字
    while (1)                       // 主循环
    {
        keynum = matrixkey();       // 调用矩阵键盘扫描函数
        if (keynum != 0) {          // 若有按键按下
            if (keynum <= 10)       // 若按键编号在1-10范围内，是输入密码区域
            {
                if (count < 4)      // 如果已输入位数小于4
                {
                    password *= 10; // 密码左移一位
                    password += keynum % 10; // 添加新输入的数字
                    count++;        // 位数加1
                }
            }
            LCD_ShowNum(2, 1, password, 4); // 在LCD第2行显示密码
        }
        if (keynum == 11) {         // 若按键11按下（确认键）
            if (password == 3456) { // 若密码正确
                LCD_ShowString(1, 14, "OK "); // 显示"OK"
                password = 0;       // 清空密码
                count = 0;          // 清空计数
                LCD_ShowNum(2, 1, password, 4); // 清空显示
            }else {                 // 若密码错误
                LCD_ShowString(1, 14, "ERR"); // 显示"ERR"
                password = 0;       // 清空密码
                count = 0;          // 清空计数
                LCD_ShowNum(2, 1, password, 4); // 清空显示
            }
        }
        if (keynum == 12) {         // 若按键12按下（清除键）
            password = 0;           // 清空密码
            count = 0;              // 清空计数
            LCD_ShowNum(2, 1, password, 4); // 清空显示
        }
    }
}
```

**代码解释：**
- `password`变量存储输入的密码，通过乘10和加法实现数字拼接
- `count`变量记录已输入的位数，限制最多输入4位
- 按键11用于确认密码，按键12用于清除密码
- 使用LCD1602显示密码和结果

### 关键知识点
1. **密码输入逻辑**：通过数字拼接实现多位密码输入
2. **状态管理**：使用变量记录输入状态和位数
3. **用户交互**：提供确认和清除功能，提高用户体验

### 遇到的问题和解决方法
**问题**：密码输入时，数字显示不正确。

**解决**：检查密码拼接逻辑，确保：
1. `password *= 10`正确左移一位
2. `password += keynum % 10`正确添加新数字
3. `count`变量正确递增
4. 输入位数不超过4位
