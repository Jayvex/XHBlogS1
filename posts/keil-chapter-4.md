## 引言

数码管是单片机常用的显示设备，可以显示数字和部分字符。本章将学习：
- 数码管的工作原理
- 静态数码管显示方法
- 动态数码管扫描显示技术

本章将从静态数码管显示开始，学习动态数码管的扫描显示原理。

## 项目列表

1. **4-1 静态数码管显示**：学习数码管的基本显示方法
2. **4-2 动态数码管显示**：掌握动态扫描显示技术

## 项目1：4-1 静态数码管显示

### 项目目标
在数码管上静态显示数字，学习数码管的基本显示方法。

### 代码解析
```c
//CPU: AT89C52
//Freq: 12MHz

#include <reg52.h>  // 包含51单片机寄存器定义头文件

// 数码管显示码表，对应0-8的段码
unsigned char nixie_table[] = {0x3f, 0x06, 0x5b, 0x4f, 0x66, 0x6d, 0x7d, 0x07, 0x7f};

// 数码管显示函数
// location: 位置（1-8）
// num: 要显示的数字（0-8）
void nixie(unsigned char location, unsigned char num)
{
    // 根据位置选择数码管（位选）
    switch(location)
    {
        case 1:
            P24 = 1;P23 = 1;P22 = 1;  // 选择第1个数码管
            break;
        case 2:
            P24 = 1;P23 = 1;P22 = 0;  // 选择第2个数码管
            break;
        case 3:
            P24 = 1;P23 = 0;P22 = 1;  // 选择第3个数码管
            break;
        case 4:
            P24 = 1;P23 = 0;P22 = 0;  // 选择第4个数码管
            break;
        case 5:
            P24 = 0;P23 = 1;P22 = 1;  // 选择第5个数码管
            break;
        case 6:
            P24 = 0;P23 = 1;P22 = 0;  // 选择第6个数码管
            break;
        case 7:
            P24 = 0;P23 = 0;P22 = 1;  // 选择第7个数码管
            break;
        case 8:
            P24 = 0;P23 = 0;P22 = 0;  // 选择第8个数码管
            break;
    }
    P0 = nixie_table[num];  // 输出段码（段选）
}

void main()
{
    nixie(8, 8);  // 在第8个数码管显示数字8
    while (1)
    {
        // 空循环，保持显示
    }
}
```

**代码解释：**
- `nixie_table[]`：数码管段码表，存储0-8对应的段码值
- `nixie()`函数：实现数码管显示，包含位选（选择数码管）和段选（显示数字）
- `P24、P23、P22`：位选信号，通过3个引脚选择8个数码管中的一个
- `P0`：段选信号，输出数字对应的段码

### 关键知识点
1. **数码管结构**：了解共阴极和共阳极数码管的区别
2. **位选和段选**：位选选择哪个数码管，段选显示什么数字
3. **段码表**：预先计算好每个数字对应的段码值

### 遇到的问题和解决方法
**问题**：不理解段码表的计算方法。

**解决**：数码管有8个段（a-g和dp），每个段对应一个位。例如，数字0需要点亮a、b、c、d、e、f段，熄灭g和dp段，对应的二进制为00111111，即0x3f。可以通过查表或手动计算得到每个数字的段码。

## 项目2：4-2 动态数码管显示

### 项目目标
实现动态数码管扫描显示，掌握动态显示技术。

### 代码解析
```c
//CPU: AT89C52
//Freq: 12MHz

#include <reg52.h>  // 包含51单片机寄存器定义头文件

// 1毫秒延时函数，12MHz晶振
void delay(unsigned int xms)
{
    unsigned char i, j;
    while (xms--)
    {
        i = 2;
        j = 239;
        do
        {
            while (--j);
        } while (--i);
    }
}

// 数码管显示码表，对应0-8的段码
unsigned char nixie_table[] = {0x3f, 0x06, 0x5b, 0x4f, 0x66, 0x6d, 0x7d, 0x07, 0x7f};

// 数码管显示函数
// location: 位置（1-8）
// num: 要显示的数字（0-8）
void nixie(unsigned char location, unsigned char num)
{
    // 根据位置选择数码管（位选）
    switch(location)
    {
        case 1:
            P24 = 1;P23 = 1;P22 = 1;  // 选择第1个数码管
            break;
        case 2:
            P24 = 1;P23 = 1;P22 = 0;  // 选择第2个数码管
            break;
        case 3:
            P24 = 1;P23 = 0;P22 = 1;  // 选择第3个数码管
            break;
        case 4:
            P24 = 1;P23 = 0;P22 = 0;  // 选择第4个数码管
            break;
        case 5:
            P24 = 0;P23 = 1;P22 = 1;  // 选择第5个数码管
            break;
        case 6:
            P24 = 0;P23 = 1;P22 = 0;  // 选择第6个数码管
            break;
        case 7:
            P24 = 0;P23 = 0;P22 = 1;  // 选择第7个数码管
            break;
        case 8:
            P24 = 0;P23 = 0;P22 = 0;  // 选择第8个数码管
            break;
    }
    P0 = nixie_table[num];  // 输出段码（段选）
}

void main()
{
    while (1)
    {
        // 动态扫描显示，依次显示1、2、3
        nixie(1, 1);  // 在第1个数码管显示数字1
        delay(500);   // 延时500ms
        nixie(2, 2);  // 在第2个数码管显示数字2
        delay(500);   // 延时500ms
        nixie(3, 3);  // 在第3个数码管显示数字3
        delay(500);   // 延时500ms
    }
}
```

**代码解释：**
- `delay()`函数：提供毫秒级延时，用于控制扫描速度
- 动态扫描：依次点亮每个数码管，利用人眼视觉暂留效应
- 循环显示：在主循环中依次显示不同数字

### 关键知识点
1. **动态扫描原理**：利用人眼视觉暂留效应，快速切换显示内容
2. **扫描速度**：扫描速度需要足够快，通常每个数码管显示1-5ms
3. **消影处理**：切换数码管时需要先熄灭所有数码管，避免残影

### 遇到的问题和解决方法
**问题**：数码管显示有残影或闪烁。

**解决**：这是因为扫描速度不够快或消影处理不当。可以：
1. 减少每个数码管的显示时间（如从500ms减少到5ms）
2. 在切换数码管前先熄灭所有数码管（P0=0x00）
3. 使用定时器中断实现更稳定的扫描
